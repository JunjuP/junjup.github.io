Co-op Top Down 3D Shooter

This Helldivers-inspired cooperative shooter is not yet close to be a complete game, but will be improved with new features in the upcoming months!

Project highlights:

  • Dynamic camera: it reacts to the distance between players
  • Friendly Fire
  • Multiple player inputs (up to four!)

//Function to get the centroid of the positions of the players
func get_centroid(p_positions):
  var px :float = 0.0;
  var pz :float = 0.0;
  var py :float = 0.0;
  var cnt = 0;

  var max_distance = 0.0;

  for pos in p_positions:
   cnt += 1;
   px += pos.x;
   pz += pos.z;
   for p in p_positions:
    if p.distance_to(pos) > max_distance:
     max_distance = p.distance_to(pos);
   if max_distance > 15.0:
    py += max_distance-15.0;
    py = clamp(py, py, 5.0)

  px = px/cnt;
  pz = pz/cnt;

  return Vector3(px-7.0, camera_position.y + py, pz)
